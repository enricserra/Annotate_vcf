Add information contained in VCF files to a main vcf file, in matching variants.

   Usage:
    
     python Annotate_vcf.py -i <INPUT_VCF> -d [ANNOTATION_VCF_LIKE_FILE1_ ANNOTATION_VCF_LIKE_FILE_2 ... ANNOTATION_VCF_LIKE_FILE_N ] -o <OUTPUT_FILE> [-m True/False]


   -m False will not merge vcf headers.
